# Docs - Dokümantasyon Klasörü

Bu klasör, ObservAI projesine ait teknik dokümantasyon ve kılavuzları içerir.

## İçindeki Dosyalar

### CAMERA_SOURCES_GUIDE.md
Kamera kaynaklarını yapılandırma kılavuzu:
- Webcam kullanımı
- IP kamera bağlantısı
- Video dosyası işleme
- RTSP stream desteği

### ECHARTS_USAGE_GUIDE.md
ECharts kütüphanesi kullanım kılavuzu:
- Grafik oluşturma
- Tema özelleştirme
- Responsive tasarım
- Real-time data güncelleme

### PROJECT_SUMMARY.md
Projenin genel özeti:
- Mimari yapı
- Kullanılan teknolojiler
- Ana özellikler
- Proje hedefleri

## Ne İçin Kullanılır?

Bu klasör şunlar için kullanılır:
- **Teknik Dokümantasyon** - Geliştiriciler için detaylı kılavuzlar
- **Kullanım Örnekleri** - Kod örnekleri ve best practice'ler
- **Konfigürasyon Rehberleri** - Sistem yapılandırma adımları
- **Referans Materyaller** - API ve kütüphane referansları

## Yeni Doküman Ekleme

Bu klasöre yeni doküman eklerken:
1. Markdown (.md) formatı kullanın
2. Açıklayıcı başlık ve içindekiler tablosu ekleyin
3. Kod örneklerini syntax highlighting ile yazın
4. Görseller varsa `docs/images/` klasörüne ekleyin

## Notlar

- Genel kurulum kılavuzu ana dizindeki `README.md` dosyasındadır
- Frontend spesifik dokümantasyon `frontend/README.md` içindedir
- Backend API dokümantasyonu `backend/README.md` içindedir
- Python analytics dokümantasyonu `packages/camera-analytics/README.md` içindedir
